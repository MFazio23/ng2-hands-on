##Install _Angular CLI_

<img src="img/angular.svg" height="300" />

_npm install -g angular-cli_

Note:
+ Can use _ng generate_ or _ng g_
    + Components
    + Directives
    + Pipes
    + Services
    + Classes
    + Interfaces
    + Enums
    + Modules
+ _ng build_
    + Build artifacts moved to _dist_ directory
